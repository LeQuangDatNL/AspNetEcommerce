# Dự án Quản lý Bán hàng

## 🏗 Kiến trúc dự án
Dự án được chia thành các tầng (N-Tier):
* **Admin/Shop**: Giao diện người dùng.
* **BusinessLayers**: Xử lý nghiệp vụ.
* **DataLayers**: Thao tác dữ liệu.
* **Models**: Thực thể dữ liệu.

## 🛠 Công nghệ
* C#, .NET, SQL Server.
* Entity Framework, ASP.NET MVC, Razor Pages.
* HTML, CSS, JavaScript, jQuery, Adminlte bootstrap 5.

## 🚀 Tính năng chính (dự kiến))

* Quản lý sản phẩm: Thêm, sửa, xóa sản phẩm.
* Quản lý loại sản phẩm: Thêm, sửa, xóa loại sản phẩm.
* Quản lý khách hàng: Thêm, sửa, xóa khách hàng.
* Quản lý nhà cung cấp: Thêm, sửa, xóa nhà cung cấp.
* Quản lý nhân viên: Thêm, sửa, xóa nhân viên.
* Quản lý đơn hàng: Tạo, xem, cập nhật trạng thái đơn hàng.
* Tìm kiếm sản phẩm: Tìm kiếm sản phẩm theo tên, loại, giá.
* Quản lý người dùng: Thêm, sửa, xóa người dùng, phân quyền truy cập.
* Giỏ hàng: Thêm sản phẩm vào giỏ, cập nhật số lượng, xóa sản phẩm khỏi giỏ, thanh toán đơn hàng.
